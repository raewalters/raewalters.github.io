Off Campus Library Services (OCLS) is equipped to meet the information needs of the typical student. However, if time is short, or an extensive selection of print materials desired, consult libraries in your vicinity. Links to several library directories are listed below.

  • Indiana State Library - Catalog lists available library services.
Illinois

Kentucky
  • Kentucky Virtual Library - Kentucky library databases (library card required).
  • Kentucky Library Directory - From the Department for Libraries and Archives.
Ohio
  • Ohio Public Library Websites - Directory of Ohio public libraries.
  • OHIOLINK - Catalog of academic libraries in Ohio.

Asia

World Wide
  • Worldwide Directory of Libraries - Directory of major libraries around the world.
  • WorldCat Library Catalog - Indexes large libraries around the globe.
Digital
  • Google Books - Selected pages from millions of books.
  • Project Gutenberg - Over forty thousand public domain (pre-1922) books.
